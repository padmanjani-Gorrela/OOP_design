# Pizza Generator 🍕

A simple Java mini project based on Object-Oriented Programming (OOP) concepts.

## Overview

This project simulates a basic pizza ordering and billing system where users can:

* Create Veg or Non-Veg pizzas
* Add extra cheese
* Add extra toppings
* Opt for takeaway packaging
* Generate the final bill

The project is designed to practice core OOP principles like:

* Classes and Objects
* Constructors
* Encapsulation
* Methods and State Management
* Package Structure in Java
